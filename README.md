# JobMap

A TypeScript application for job mapping and management.

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Building

```bash
npm run build
```

### Running

```bash
npm start
```

### Development

```bash
npm run dev
```

### Testing

```bash
npm test
```

## Project Structure

```
jobmap/
├── src/           # Source files
├── dist/          # Compiled output
├── package.json   # Project configuration
└── tsconfig.json  # TypeScript configuration
```
